For installing

```
  $ mkdir -p $ZSH_CUSTOM/themes && curl https://raw.githubusercontent.com/VGamezz19/oh-my-zsh-fortuity-theme/master/fortuity.zsh-theme -L -o $ZSH_CUSTOM/themes/fortuity.zsh-theme
```

Set the theme in your **.zshrc** file 

```
  ZSH_THEME="fortuity"
```
